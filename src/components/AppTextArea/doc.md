## APP TextArea Component

Based on Bootstrap text area.

**Required Props**
|Prop |Value(s) |
|--|--|
| name | string |
| rows | number |

**Optional Props**
| Prop | Value(s) |
|--|--|
| id | string |
| label | string |
| required | true or false |
| placeholder | string |
| value | string |
| disabled | true or false |

**Examples**

1.  Simple Text area .

        <AppTextArea name="test" label="Test Input" rows="3"/>
